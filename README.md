## 🔧 Basic Troubleshooting

If the app does not start, try these steps:

- Check that the file finished downloading
- Extract the zip file again
- Run the file as administrator
- Make sure Docker Desktop is running
- Restart Windows and try again
- Check the `logs` folder for the latest message

If the window opens and closes fast, the app may be missing a required file. Open the folder and start it from the main launch file again.
